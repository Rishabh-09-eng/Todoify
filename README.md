# Todoify

A responsive Todo application built with React and Tailwind CSS.

## Features

- Add Todos
- Edit Todos
- Delete Todos
- Mark Todos as Completed
- Show/Hide Finished Todos
- Local Storage Persistence
- Mobile Responsive Design
- Keyboard Support (Enter Key)

## Tech Stack

- React
- Tailwind CSS
- React Icons
- UUID
- Local Storage

## Installation

```bash
npm install
npm run dev
